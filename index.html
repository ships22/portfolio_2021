What I do

Responsive design

Responsive design is a key component of the web development process. The guidelines for responsive design ensure that HTML, CSS (cascading style sheets), and JavaScript elements like menus, text, and buttons are clear and usable everywhere.

Web development

Web development has numerous components that work together to create the finished experience of a website, including graphic design, user experience design, interface design, search engine optimization (SEO) and content creation. These elements determine how a website looks, feels and works on various devices.

Graphic design

Graphic design is a craft where professionals create visual content to communicate messages. By applying visual hierarchy and page layout techniques, designers use typography and pictures to meet users' specific needs and focus on the logic of displaying elements in interactive designs, to optimize the user experience.

Clean code

Code is clean if it can be understood easily – by everyone on the team. Clean code can be read and enhanced by a developer other than its original author. With understandability comes readability, changeability, extensibility and maintainability.

SEO marketing

Search engine optimization (SEO) is the practice of orienting your website to rank higher on a search engine results page (SERP) so that you receive more traffic. The aim is typically to rank on the first page of Google results for search terms that mean the most to your target audience.

Great support

Tech support is probably one of the most unloved jobs in technology. Developers frequently look down on tech support as people who can’t program. I always try to be there when you need me.
